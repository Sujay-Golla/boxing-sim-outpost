# Boxing Simulator
By: Sujay Golla, Arush Sastry, Sreekar

## Inspiration
The inspiration for this project partially comes from VR experience, which uses body sensors to let users experience and interact with a different reality. Combined with our shared interests for combat sports, we chose to use robotic arms to simulate a boxing game and create a fun and interactive demo for Open Sauce.

## Challenges
The time constraint was the biggest challenge in designing, prototyping and refining our project. Since it was a hackathon, a lot of the wiring and 3D models were initial iterations and had to be adapted to create a working demo. Initially we were planning to make it a two-player interactive game but we had to scope it down to accomodate for the materials and timing constraints to a one-player score-based boxing game.

## Technical Features
The project involves a pair of 3-DOF robotic arms controlled by MPU-6050 IMU sensors in the form of an armband that users can wear to interact the board. Depending on the complexity and execution of a boxing move, players can score points and finish objectives. In a two-player extension, a boxing spar would take place, where the player with the most points would win.
